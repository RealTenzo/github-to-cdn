# GitHub to CDN

Convert raw GitHub URLs to fast CDN links in one click.
Built with **HTML, CSS, and JavaScript**.

**Live Demo:** [https://github-to-cdn.vercel.app](https://github-to-cdn.vercel.app)

---

## Features

* Convert GitHub raw URLs to CDN URLs instantly
* Copy or open the CDN link with a click
* Dark mode and light mode support
* Mobile-friendly responsive design
* Small and fast, no frameworks required

---

## Usage

1. Open the live demo or your local copy.
2. Paste a **GitHub Raw URL**, for example:

   ```
   https://raw.githubusercontent.com/user/repo/branch/file.js
   ```
3. Click **Convert** or press `Enter`.
4. Copy the generated CDN link or open it in a new tab.

---

## CDN Example

| GitHub Raw URL                                               | CDN URL                                                |
| ------------------------------------------------------------ | ------------------------------------------------------ |
| `https://raw.githubusercontent.com/user/repo/branch/file.js` | `https://cdn.jsdelivr.net/gh/user/repo@branch/file.js` |

---

## Dark Mode

Use the toggle button in the top-right corner to switch between light and dark mode.
Your preference is saved using `localStorage`.

---

## Installation / Hosting

No installation is required.

You can open `index.html` directly in a browser or host it using **GitHub Pages**.

```bash
git clone https://github.com/RealTenzo/github-to-cdn.git
cd github-to-cdn
```
