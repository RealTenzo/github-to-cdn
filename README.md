# GitHub to CDN

Convert raw GitHub URLs to fast CDN links in one click!
Built with **HTML, CSS, and JavaScript**.

## Features

- Convert GitHub raw URLs to CDN URLs instantly.
- Copy or open the CDN link with a click.
- Dark mode and light mode support.
- Mobile-friendly responsive design.
- Small and fast: no frameworks required.

## Usage

1. Open the web page.
2. Paste a **GitHub Raw URL** (e.g., `https://raw.githubusercontent.com/user/repo/branch/file.js`) into the input box.
3. Click **Convert** or press `Enter`.
4. Copy the generated CDN link or open it in a new tab.

## CDN Example

| GitHub URL | CDN URL |
|------------|---------|
| `https://raw.githubusercontent.com/user/repo/branch/file.js` | `https://cdn.jsdelivr.net/gh/user/repo@branch/file.js` |
